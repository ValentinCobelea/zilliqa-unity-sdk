Detailed Description

This password hashing system tries to thwart off-line password cracking using a computationally-intensive hashing algorithm, the work factor of the algorithm is parameterised, so it can be increased as computers get faster. See http://www.tarsnap.com/scrypt/scrypt.pdf for algorithm details.
